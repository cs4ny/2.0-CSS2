Procedural Programming is one of the first course in CS20. Main concepts include the ability to organize codes into block that are easy to navigate and read. Starting with functions that can encode similar codes together to be used repeatedly, procedural programming has proven to be extremely useful in text based programming language such as with Python. However, dues to JavaScript's main uses of global variables, procedural programming can be challenging in intergrating it in an interactive website.

Above is a Rock Paper Scissors game that was translated from Python to JavaScript. Though it does not include all of Python code, the JavaScript program is more interactive with its user. Click on either Rock, Paper or Scissors pictures above and see what happens.
